## Todo test api

**Stack:**
- Express.js
- MongoDB
- Mongoose
- Graphql

## How to run

1. npm install

2. add .env file to the root

3. npm run seed (Optional to add a test Admin user)

4. npm run dev

## Test url after runnig the server
``` 
 http://localhost:5000/graphql
 ```
